<h1 align="center">newsRec</h1>




<div align="center">

News recommendation system based on nodejs,vue3;基于vue3,nodejs 的新闻推送系统


</div>

# 目录

newsRec:后端nodejs,

newsRec-vue3:前端vue3

# 部署

在各端目录中运行

```
npm install
```

# 运行

在各端目录中运行

```
npm run dev
```
